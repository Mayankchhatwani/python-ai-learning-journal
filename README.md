# Python AI Learning Journal

This repository tracks my daily progress as I learn:

- Intermediate → Advanced Python
- Machine Learning & Deep Learning
- LLMs and Agentic AI systems

## Structure

- `daily-progress/` – Day-wise logs (what I learned, code, doubts)
- `concepts/` – Short notes on each concept
- `challenges/` – Coding exercises & solutions
- `mini-projects/` – Small weekly/bi-weekly projects
- `ml-notebooks/` – Jupyter notebooks for ML experiments
- `roadmap/` – Overall learning roadmap
- `resources/` – Useful links & references
- `weekly-reviews/` – Reflection & progress review
